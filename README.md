
# 🚀 Real-Time Code Editor

A real-time collaborative code editor built using **React, Node.js, Express, and Socket.io** that allows multiple users to write and edit code simultaneously.

---

## 📌 Project Overview

This project enables multiple users to join a shared coding room and edit code in real-time. Any changes made by one user are instantly reflected for all connected users using WebSocket communication.

It demonstrates real-time communication, frontend-backend integration, and collaborative system design.

## ▶️ How to Run Locally

### 1️⃣ Clone the Repository


git clone https://github.com/RiteshMelkani/realtime-code-editor.git


### 2️⃣ Install Dependencies


npm install


### 3️⃣ Start Backend Server


node server.js


### 4️⃣ Start Frontend

If using React:


npm start
